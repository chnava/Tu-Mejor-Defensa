## 🚀 Instalación y Uso

### Requisitos Previos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Servidor web local (opcional, pero recomendado)

### Pasos de Instalación

1. **Clonar el repositorio**
```bash
git clone https://github.com/TU_USUARIO/Tu-mejor-Defensa.git
cd Tu-mejor-Defensa
```

2. **Abrir el sitio**
```bash
# Opción A: Abrir directamente
open index.html

# Opción B: Usar Live Server (VS Code)
# Instalar extensión "Live Server" y clicar "Go Live"

# Opción C: Python HTTP Server
python -m http.server 8000
# Abrir http://localhost:8000
```

3. **Personalizar (opcional)**
- Editar `index.html` para contenido
- Modificar `styles.css` para estilos
- Ajustar `script.js` para funcionalidad

## 🌐 Despliegue

### Opciones de Hosting

#### 1. GitHub Pages (Recomendado)
```bash
# El sitio ya está configurado para GitHub Pages
# Solo habilitar en Settings > Pages
```

#### 2. Netlify
- Arrastrar carpeta a [Netlify Drop](https://app.netlify.com/drop)
- O conectar repositorio GitHub

#### 3. Vercel
```bash
npm i -g vercel
vercel
```

#### 4. Hosting Tradicional
- Subir archivos vía FTP a cualquier hosting
- Compatible con Apache, Nginx, IIS

### Configuración de Dominio

Para dominio personalizado `tumejordefensa.mx`:
1. Configurar DNS en proveedor de dominio
2. Apuntar A record a IP del hosting
3. Configurar CNAME para www
